# ❤️ Whenever You Miss Me

A small website made with love to help someone special feel a little closer, no matter the distance.

Whenever you need me, I'm only one click away. ❤️

## ✨ Features

* 💙 Personalized voice messages
* 🌅 Good Morning message
* 🌙 Good Night message
* 😨 Comfort message for anxious moments
* 🌟 Encouragement and support
* 💋 Lots of kisses
* 🎵 Fixed audio player that appears while a recording is playing
* 📱 Mobile-friendly design

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
